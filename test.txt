Запуск сервера
uvicorn main:app --reload

Вход в документацию API для тестов
http://localhost:8000/docs

Создание нового рецепта
POST /recipes/

{
   "title": "Омлет",
    "cooking_time": 10,
    "ingredients": ["яйца", "молоко", "соль"],
    "description": "Взбить яйца с молоком, посолить и пожарить на сковороде."
}

Создание второго рецепта (для проверки сортировки)
POST /recipes/

{
   "title": "Борщ",
    "cooking_time": 60,
    "ingredients": ["свекла", "капуста", "мясо"],
    "description": "Сварить бульон, добавить овощи..."
}

Получение списка рецептов
GET /recipes/

Получение детальной информации о рецепте – первый просмотр
GET /recipes/{id}

Повторный просмотр того же рецепта – увеличение счётчика
GET /recipes/{id}

 Проверка сортировки после увеличения просмотров
GET /recipes/

 Запрос несуществующего рецепта
GET /recipes/{id}
